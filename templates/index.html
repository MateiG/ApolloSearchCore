What is ApolloSearch?

ApolloSearch is a tool that lets you search any PDF using natural language (like Googling it).

Each page takes about 3 seconds to index. Please wait for the entire document to finish processing.

The upload limit is 100 MB, with no limit on number of pages. To delete everything you've uploaded, click here.
